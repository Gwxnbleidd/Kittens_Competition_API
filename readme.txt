Чтобы запустить API на пк, нужно перейти в директорию doker_dir/app, далее нужно установить все зависимости.
Для этого нужно открыть терминал в этой папке и прописать pip install -r requirements.txt
Далее нужно просто запустить main.py файл. 
Сделать это можно также в терминале с помощью python3 main.py или python main.py

Чтобы запустить это в docker-compose, нужно зайти в директорию docker_dir и в терминале прописать docker-compose up.
